# Teknoloji Yığını (Tech Stack) - Trace-Impact

Trace-Impact projesinde kullanılan ana teknolojiler, kütüphaneler ve geliştirme araçları aşağıda listelenmiştir:

### 1. Frontend (Arayüz)
* **Framework:** Next.js / React (Modern ve hızlı bir kullanıcı deneyimi için).
* **Styling:** Tailwind CSS (Responsive ve şık bir tasarım için).
* **Veri Görselleştirme:** Chart.js / Lucide-React (Karbon ayak izi ve regresyon tahmin grafiklerini sunmak için).

### 2. Backend & Analiz (Arka Plan)
* **Dil:** Python / JavaScript (Hesaplama algoritmaları için).
* **Tahminleme Modeli:** Scikit-learn tabanlı Regresyon Analizi (Gelecek projeksiyonları oluşturmak için).
* **Hata Yönetimi:** Offline Fallback Regression Model (Bağlantı koptuğunda tarayıcı tabanlı hesaplamaya geçiş).

### 3. Altyapı & Geliştirme (Infrastructure)
* **Geliştirme Ortamı:** Cursor AI (AI destekli kod yazımı için).
* **Yayınlama (Deployment):** Vercel (Hızlı ve güvenilir canlıya alma süreci).
* **Versiyon Kontrol:** GitHub (Sürüm yönetimi ve iş birliği).

### 4. Yapay Zeka Entegrasyonu
* **Model:** OpenAI API (Kullanıcı verilerini yorumlamak ve kişiselleştirilmiş öneriler sunmak için).
